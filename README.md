# Hotel Estrella Real

## Descripción
Sistema de gestión de reservas para hotel desarrollado con Java y Spring Boot. Permite a los usuarios buscar habitaciones, hacer reservas, gestionar pagos y recibir confirmaciones por correo electrónico.

## Tecnologías utilizadas
- Java 17  
- Spring Boot  
- Hibernate / JPA  
- MySQL (o la base de datos utilizada)  
- Thymeleaf (para la interfaz web, si aplica)  
- Maven / Gradle  
- Git  

## Características principales
- Registro e inicio de sesión de usuarios  
- Búsqueda de habitaciones por fecha y tipo  
- Gestión de reservas y cancelaciones  
- Pago en línea (simulado o integración real)  
- Administración de habitaciones y precios  
- Envío de correos electrónicos de confirmación
